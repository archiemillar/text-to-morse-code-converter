<h1 align="center"> Text to Morse Code Converter </h1>
<p align="center">A text-based Python program to convert Strings into Morse Code.</p>


### Requirements:
- `numpy`
- `pandas`
- `prettify`
- `prettytable`
- `pyfiglet`
- `python-dateutil`
- `pytz`
- `six`
- `tabulate`
- `cwidth`

# Setup:
```
# clone repository & change directory
git clone --depth=1 https://github.com/archiemillar/text-to-morse-code-converter.git

# install dependency libraries
pip install -r requirements.txt
```

# Usage:
```
# Run
python main.py
```
